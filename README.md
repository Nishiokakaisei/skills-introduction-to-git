# Git 入門

_コマンドラインと VS Code を使い、Git のバージョン管理で小さなゲームを開発します。_

## ようこそ

- **対象者**: Git のバージョン管理を初めて学ぶ開発者
- **学ぶこと**: コミット、ブランチ、履歴、共同作業の基本
- **作るもの**: サンプルゲームのリポジトリを作成し、いくつかの機能を追加します
- **前提知識**:

  - Git やバージョン管理の経験は不要です。
  - 推奨: コマンドラインインターフェイス (CLI) の基本操作
  - 推奨: Visual Studio Code の基本操作

- **所要時間**: 60 分以内で完了できます。

この演習では、次のことを行います。

1. バージョン管理とは何か、なぜ開発者が使うのかを理解する。
1. Git のユーザー情報を設定する。
1. はじめてのリポジトリを作成し、コミットする。
1. プロジェクト履歴を確認し、ファイルの差分を比較する。
1. ブランチを使って安全に実験する。
1. Git による共同作業の考え方を学ぶ。
